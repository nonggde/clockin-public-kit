# CLOCKIN Public Roadmap

CLOCKIN is product-first. The token, if launched, should point at real skill utility rather than promise utility.

## Live Now

- Public skill console: https://clockin-agent.pages.dev
- Public `SKILL.md` workflow file.
- Skill catalog and detail pages.
- Opportunity Scan public-input demo.
- Draft Skill generator.
- No-signature Creator Pass preview for public Solana addresses.
- Agent Card and launch metadata.
- Public safety boundaries.

## Next Milestones

1. Collect skill route requests through GitHub Issues.
2. Add stored creator profile flow after a database or GitHub-backed queue exists.
3. Add visible wallet connect and signed ownership message.
4. Turn accepted skill requests into public skill pages.
5. Test access-pass utility for skill packs, listing actions, and route voting.

## Not On The Roadmap

- Profit promises.
- Revenue share promises.
- Buyback promises.
- Fake engagement.
- Wallet actions without visible user approval.
- Private key, seed phrase, password, or one-time-code handling.
