# CLOCKIN Safety Boundary

CLOCKIN is designed for public-input AI workflows.

## Allowed

- public URLs
- public repositories
- user-provided text
- product summaries
- workflow ideas
- public documentation
- public wallet addresses for format-only creator pass previews

## Blocked

- passwords, API keys, cookies, tokens, seed phrases, private keys, one-time codes
- private dashboards or private repositories without explicit access
- wallet signing or transactions
- wallet ownership or balance claims from a public address alone
- fake engagement, spam, impersonation, wash trading, or market manipulation
- account buying, account selling, verification-code tasks, or gray-area account operations
- guaranteed income, investment returns, revenue share, buybacks, or price targets

## Manual Approval Required

- paid API calls
- final public submissions
- wallet connections or signatures
- token creation
- publishing anything that claims project status, launch status, CA, or payment terms

## Creator Pass Boundary

Creator Pass accepts public Solana addresses only as public identifiers. It does not connect a wallet, request a signature, verify ownership, check balances, or transfer tokens.
