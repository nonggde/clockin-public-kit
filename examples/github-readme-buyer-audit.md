# Example: Public GitHub README Buyer Audit

## Skill

Public Code Review / Opportunity Scan

## Public Input

```text
https://github.com/nonggde/clockin-public-kit
```

## Prompt

```text
Audit this public GitHub repo as a buyer or builder:
https://github.com/nonggde/clockin-public-kit

Return:
1. What the project appears to do
2. What a stranger can try in under 2 minutes
3. Missing proof or unclear claims
4. Three GitHub issue ideas
5. README copy improvements
6. Safety boundary
```

## Expected Artifact

- Project clarity summary.
- Fast-start test path.
- Missing trust proof.
- Three concrete issue ideas.
- README improvement bullets.

## Safety Boundary

Public repository only. No private source, secrets, exploit instructions, credential handling, or destructive actions.
